# PRD - 네이버 뉴스 블로그 포스팅 CLI

## 개요

키워드를 입력하면 네이버 뉴스를 자동으로 수집·분석하고, Gemini AI가 블로그 포스트를 생성하여 네이버 블로그에 자동으로 포스팅하는 CLI 도구.

---

## 기술 스택

| 역할 | 기술 |
|------|------|
| 언어 | Node.js + TypeScript |
| 뉴스 수집 | 네이버 검색 API (공식) |
| AI 분석/요약 | Google Gemini API (무료 티어) |
| 블로그 포스팅 | 네이버 블로그 API (OAuth 2.0) |

---

## 실행 흐름

```
$ blog-post
> 키워드를 입력하세요: 인공지능
> 포스트 스타일을 선택하세요: [1] 뉴스레터  [2] 칼럼
  ↓
네이버 뉴스 최신 10개 수집
  ↓
Gemini로 분석 → 블로그 포스트 생성 (1,000~1,500자)
  ↓
[미리보기] 제목 / 본문 출력
> 포스팅할까요? (y/n):
  ↓ y
네이버 블로그 전체공개로 포스팅
  ↓
[완료] 제목 / 태그 / URL 출력
```

---

## 기능 요구사항

### 1. 키워드 입력
- CLI 실행 후 프롬프트로 키워드 입력 받음

### 2. 포스트 스타일 선택
- **뉴스레터 스타일**: 뉴스별로 제목 + 2~3줄 요약을 리스트 형태로 나열
- **칼럼 스타일**: 뉴스를 맥락으로 엮어 서론 → 본론 → 결론 구조의 하나의 글로 작성

### 3. 뉴스 수집
- 네이버 검색 API로 키워드 기준 최신 뉴스 10개 수집

### 4. AI 분석 및 블로그 포스트 생성
- Gemini API로 수집한 뉴스를 분석
- 선택한 스타일에 맞게 포스트 생성
- 분량: 1,000~1,500자
- 블로그 제목, 본문(HTML), 태그 5개 생성

### 5. 포스팅 전 미리보기 및 확인
- 생성된 제목과 본문을 터미널에 출력
- "포스팅할까요? (y/n)" 확인 후 포스팅
- n 선택 시 종료

### 6. 네이버 블로그 포스팅
- 네이버 OAuth 2.0 인증 (최초 1회, 이후 토큰 로컬 저장)
- 공개 설정: 전체 공개

### 7. 완료 출력
- 포스팅 완료 후 터미널에 출력:
  - 블로그 포스트 제목
  - 태그 목록
  - 포스팅된 블로그 URL

## 필요한 환경 변수 (.env)

```
NAVER_CLIENT_ID=          # 네이버 검색 API용
NAVER_CLIENT_SECRET=      # 네이버 검색 API용
NAVER_BLOG_CLIENT_ID=     # 네이버 블로그 OAuth용
NAVER_BLOG_CLIENT_SECRET= # 네이버 블로그 OAuth용
GEMINI_API_KEY=           # Google AI Studio에서 무료 발급
```

---

## 명령어

```bash
blog-post          # 메인 실행 (키워드 입력 → 포스팅)
blog-post logout   # 저장된 네이버 OAuth 토큰 삭제
```
