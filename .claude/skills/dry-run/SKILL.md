키워드: $ARGUMENTS

실제 네이버 블로그 포스팅 없이 뉴스 수집 → AI 생성 → 미리보기까지만 실행하는 드라이런을 수행하세요.

## 실행 단계

1. **키워드 확인**
   - `$ARGUMENTS`가 비어있으면 사용자에게 키워드를 입력받으세요.

2. **뉴스 수집 테스트**
   - `src/services/naverNews.ts`의 `searchNews()` 구현 상태를 확인하세요.
   - 구현되어 있으면 `npm run dev` 로 실제 호출합니다.
   - 미구현(`throw new Error('Not implemented')`) 상태면 "Phase 2 미완성" 메시지를 출력하고 중단합니다.

3. **AI 포스트 생성 테스트**
   - `src/services/gemini.ts`의 `generateBlogPost()` 구현 상태를 확인하세요.
   - 구현되어 있으면 수집한 뉴스로 포스트를 생성합니다.
   - 미구현이면 "Phase 3 미완성" 메시지를 출력하고 중단합니다.

4. **미리보기 출력**
   - 아래 형식으로 터미널에 출력합니다:
   ```
   ─────────────────────────────────
   [DRY RUN] 실제 포스팅은 수행되지 않습니다
   ─────────────────────────────────
   제목: {생성된 제목}
   태그: {태그1}, {태그2}, {태그3}, {태그4}, {태그5}
   ─────────────────────────────────
   {본문 전체}
   ─────────────────────────────────
   ```

5. **포스팅 단계 스킵**
   - `postToBlog()`는 호출하지 않습니다.
   - "드라이런 완료 — 포스팅 생략됨" 메시지로 종료합니다.

- 생성된 포스트 품질에 문제가 있으면 `/prompt-tuner` 에이전트 실행을 제안하세요.
